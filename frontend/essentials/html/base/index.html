什么是HTML

Html是超文本标记语言，是用来描述web文档的一种标记语言。
使用标签作为页面的开始和结束部分：

  1. html通常被称为静态网页；
  2. HTML是带html或Htm扩展名的文件；
  3. HTML的一些标签代码规则将内容呈现在浏览器中；
  4. HTML可以使用记事本创建，并以.html或.htm为扩展名保存。

HTML标签

HTML 标签是由尖括号包围的关键词，比如 <div>

HTML 标签通常是成对出现的，标签对中的第一个标签是开始标签，第二个标签是结束标签。

标签与标签之间是可以嵌套的，但先后顺序必须保持一致，如：<div> 里嵌套 <p>，那么 </p> 必须放在 </div> 的前面。

HTML文件基本结构

HTML常用标签

Head中的标签

meta 元信息

link 引入样式文件

css 文件建议放到页面顶部 </head> 前，提升页面渲染速度。

link 标签还可以用于指定网页图标。

script 引入脚本文件

js 文件建议放到页面底部 </body> 前，这样可以提示页面打开速度，不阻塞 css 渲染。

title 文档标题

title 标签必需要放在 head 标签里。

style 内联样式

Tips: style 标签建议放在 link css 下面。

这里需要说明的一个概念，一般情况下，下面的样式会覆盖上面的样式，越靠近文档底部的样式权重越高。

引入媒体资源

img 图片

video 视频

  • autoplay 是否自动播放

  • loop 是否循环

  • controls 控制按钮：播放/暂停/音量/进度/全屏

iframe 网页

意思是可以在网页再加载另外一个网页。

内容块

div 块

无意义标签，常用的布局标签。

span 行内块

无意义标签，常用的布局标签。与 div 的区别是， span 为行内标签。

p 段落

用于大篇幅文本内容的分段。不建议用于布局。

br 换行

用于大篇幅文本内容的换行。不建议用于布局。

h1~h6 标题

页面标题。

hr 水平线

ul ol 列表

table 表格

原生控件

a 超链接

form 表单

提交数据到服务端，用于登录，上传图片等。目前比较少用到这种方式了，一般使用 ajax 代替。

input 单行输入框

  • password

    密码，默认隐藏

    文件

  • number

  • radio

    单选框

  • checkbox

    复选框

textarea 多行文本输入框

select 下拉选择框

button 按钮

与 input 按钮的不同在于，button 里可以放置图片等内容。
